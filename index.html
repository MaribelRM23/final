Objetivo

Enseñar las señales informativas de manera efectiva y atractiva, facilitando su aprendizaje a los usuarios para que puedan retenerlas y aplicarlas de forma práctica

Descubre

Descubre una nueva forma de aprender las señales de manera divertida, didáctica y sencilla. Disfruta de nuestras innovadoras y creativas ideas.

Personajes

Alf

Alf

Alf es el personaje principal del juego.Este pasa por una serie de lugares confusos donde adquiere la ayuda de un libro que contiene las señales informativas, un obsequio por parte de su madre años atrás con el cuál podrá pasar cada una de las pruebas.

Val

Val

Es un personaje secundario y es la madre del personaje principal Alf. Val teme que su hijo salga porque se puede perder.Ella tiene un libro el cuál le servirá a Alf cuando se pierda.

Tadeo

Tadeo

Es un personaje secundario y el padre del personaje principal Alf.Tadeo considera que Alf no está lo suficientemente preparado y experimentado para enfrentar el reto de conocer lugares nuevos.

¿Cómo jugar?

Al iniciar partida, se mostrará a Alf en, este necesita ayuda para orientarse; el usuario utilizará las señales informativas para ayudarlo a ubicarse correctamente.En su recorrido, aparecerán diferentes señales y se presentarán preguntas con opciones múltiples relacionadas con su uso.Cuando se responda correctamente la pregunta aparecerá un letrero de felicitaciones, si el usuario se equivoca aparecerá un letrero que hará entender que se equivocó así mismo mostrará una frase para motivar al jugador.
